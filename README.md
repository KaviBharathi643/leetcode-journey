# 💡 Learning Principles

While solving problems, I follow these rules:

* Understand the problem before writing code.
* Try solving independently before checking hints.
* Spend around 30–40 minutes on a medium problem before reading the editorial.
* Learn the underlying pattern instead of memorizing solutions.
* Record mistakes for future revision.
* Revisit important problems multiple times.
